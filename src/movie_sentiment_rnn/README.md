# movie_sentiment_rnn

This package contains reusable utilities for the Movie Sentiment RNN project.

Key modules:

- `config.py` resolves environment-based project settings.
- `data.py` handles dataset validation, cleaning, and splitting.
- `text.py` contains text preprocessing helpers.
- `metrics.py` contains classification and regression metrics.
- `models.py` contains lazy TensorFlow model builders.
- `cli.py` exposes command-line utilities.
